# SoX Noise

Noise generator GUI powered by [Sound eXchange](http://sox.sourceforge.net/)

Requires: SoX, Python, & GTK3  
Install: `python -m pip install git+https://github.com/thann/sox-noise.git --user`  
Run: `sox-noise`
